# 🚀 Startup Funding Analysis Dashboard

An interactive dashboard built with Python & Streamlit to analyze 
Indian startup funding data from 2015 to 2020.

## 📊 Live Demo
[Click here to view the app](https://startup-dashboard3.streamlit.app/)

## 📌 Features
- Overview of total funding, deals & investors
- Startup wise analysis
- Investor wise analysis
- City wise analysis
- Funding trends over the years
- Interactive charts & filters
- Raw data explorer with download option

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## 📁 Dataset
- 2065 startup funding deals
- Years: 2015 to 2020
- Columns: Date, Startup, Vertical, City, Investor, Round, Amount

## 🚀 How to Run Locally
clone the repository
git clone https://github.com/rajanprasad3/startup-dashboard

install dependencies
pip install -r requirements.txt

run the app
streamlit run app.py
